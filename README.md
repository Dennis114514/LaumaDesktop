# LaumaDesktop
一个为不会使用手机的老人制作的软件  
让每一个人都能享受发展的成果  

## 功能特点：
1. 简洁且现代的设计  
软件使用Material Design3设计风格，主页简介，页面美观
2. 强大的功能和可扩展性  
软件支持通过配置文件直接编辑联系人信息，支持显示联系人图片，预留视频通话（通过无障碍操作其他应用完成）和语音播报（通过播放预录制音频或TTS）接口
3. 易于使用的逻辑  
配合我们即将发布的配置生成器，可以方便地编辑联系人信息  
配置完成后，在主页面上即可直接使用按钮一键拨号，一键拨打视频通话（待支持）  
在主界面右上角的应用抽屉内可以直接打开常用应用（由你自己设置），点击时间即可快速打开一个你设置的应用  

## 安装方法
1. 在[这里](https://github.com/Dennis114514/LaumaDesktop/releases)下载APK安装包
2. 从https://github.com/Dennis114514/LookTheWorld.git 克隆源码，然后编译
```bash
git clone https://github.com/Dennis114514/LookTheWorld.git
#Windows环境
.\gradlew build
#其他系统
gradlew build
```

## 使用方法
1. 打开应用后，按照首次安装引导进行操作。  
• 对于需要下载的三个应用，MT管理器用于编辑联系人信息，AccessibilityManager用于启动和关闭无障碍功能，QQ用于视频通话  
（编辑器可以使用其他编辑器代替，无障碍服务可以手动开关，视频通话功能尚未实装，所以都可以不安装）  
• 对于配置文件编辑，目前只支持手动编辑，按照文档进行操作即可  
2. 进入应用后请核对联系人信息是否正确，如果不正确，请点击主界面右上角的设置进行更改  
3. 应用左上角按钮为应用抽屉，点击时间区域可以快速打开预设的应用（建议设置成电视直播应用或短视频应用）
4. 应用右上角为设置，可以对应用进行配置  
5. 应用下方的电话按钮点击后可以直接拨号，无需二次确认；右侧按钮为视频通话按钮（功能尚未实装），点击后可以直接调用QQ进行视频通话
6. 应用下方的箭头可以点击来切换联系人  

## 常见问题
Q：显示“联系人信息配置错误”怎么办？  
A：  
• 如果您没有把LaumaDesktop设置为默认桌面，那么直接返回到桌面，打开MT管理器，然后依次打开/Download/LaumaDesktop文件夹，找到ContractInformation.json文件，编辑为正确的配置即可  
• 如果您已经把LaumaDesktop设置为默认桌面，可下拉通知栏，通过系统提供的快捷按钮打开“设置”应用，然后把默认启动器改回原来的桌面或者卸载本应用，改回正确的配置即可  

Q：不慎删除了README.txt文件，不会编辑了怎么办？  
A：您可以打开[https://github.com/Dennis114514/LaumaDesktop/blob/master/app/src/main/assets/README.txt]获取此文件   

Q：想打开更多的应用怎么办？  
A：点击应用右上角的“设置”，点击“全部应用”，即可查看系统安装的所有应用，点击即可打开，长按可以查看详情或卸载。  

Q：卸载重装会影响现有的配置吗？  
A：不会。我们的配置文件全部存储在Download文件夹中，这个文件夹是公有文件夹，不会随软件卸载而丢失。
