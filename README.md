# LaumaDesktop
## 功能特点：
1. 简洁且现代的设计  
软件使用Material Design3设计风格，主页简介，页面美观
2. 强大的功能和可扩展性  
软件支持通过配置文件直接编辑联系人信息，支持显示联系人图片，预留视频通话（通过无障碍操作其他应用完成）和语音播报（通过播放预录制音频或TTS）接口
3. 易于使用的逻辑  
配合我们即将发布的配置生成器，可以方便地编辑联系人信息  
配置完成后，在主页面上即可直接使用按钮一键拨号，一键拨打视频通话（待支持）  
在主界面右上角的应用抽屉内可以直接打开常用应用（由你自己设置），点击时间即可快速打开一个你设置的应用
